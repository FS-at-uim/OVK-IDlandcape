# ADR 0005: Softere ID Badge Farben
- **Status**: Accepted
- **Kontext**: Die ursprünglichen ID-Badges verwendeten sehr kräftige, gesättigte Markenfarben (z. B. leuchtendes Telekom-Rot für Utiq und kräftiges Grün für netID). Diese kräftigen Farben wirkten im Gesamtlayout unruhig und lenkten stark ab, insbesondere bei einer Vielzahl von Karten.
- **Entscheidung**: Umstellung der Badge-Farben auf ein dezenteres, moderneres Pastell-Design (heller Hintergrund mit abgetöntem, dunklerem Text in der jeweiligen Markenfarbe):
  - **Utiq**: Hintergrund `#fde8e8` (helles Rosa), Text `#c81e1e` (dunkles Rot)
  - **netID**: Hintergrund `#e6f4ea` (helles Grün), Text `#137333` (dunkles Grün)
- **Konsequenzen**:
  - **Vorteile**: Harmonischeres Gesamtbild, bessere Lesbarkeit der Texte, modernere Ästhetik.
  - **Nachteile**: Geringere Signalwirkung der einzelnen Badges auf den ersten Blick, was jedoch für die Gesamtübersicht der Karte erwünscht ist.
